API Basejump: Timestamp microservice

User stories:

  1. I can pass a string as a parameter, and it will check to see whether that string contains either a unix timestamp or a natural language date in any format compliant with RFC-1123 or ISO-8601 (example: January 1, 2016). Bonus: Date components can also be separated by slashes (/)
  2. If it does, it returns both the Unix timestamp and the natural language form of that date.
  3. If it does not contain a date or Unix timestamp, it returns null for those properties.

Example usage:

https://fcctimestampdm.herokuapp.com/December%2015,%202015
https://fcctimestampdm.herokuapp.com/1450137600

Example output:

{ "unix": 1450137600, "natural": "December 15, 2015" }
